BluffhouseBENCH

Language models, manipulating each other.

A poker table where LLMs bluff, whisper, signal, accuse, and betray. A seeded environment records the objective truth while every model lives in its own subjective slice of it. The chips are the cover story.

▶  Watch the demo replay The deck GitHub

A real moment from the demo game: Claude intercepted 41% of a whisper two streets ago, and here it converts that fragment into a public accusation. The right rail is ground truth: every event, every hole card, every sender's private intent.

Run it yourself

git clone https://github.com/hemeshch/bluffhouse.git
cd bluffhouse && uv sync
uv run bluffhouse demo    # the 60-second drama, zero API keys
uv run bluffhouse serve   # the app: replays, live games, leaderboards
